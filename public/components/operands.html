AND Operator

The AND operator, denoted by &&, is a logical operator that returns true only if both of its operands are true. Otherwise, it returns false.
Example: If both condition A and condition B are true, A && B evaluates to true; otherwise, it evaluates to false.

{{end}} {{define "or-operator"}}

OR Operator

The OR operator, denoted by ||, is a logical operator that returns true if at least one of its operands is true. It returns false only if both operands are false.
Example: If either condition A or condition B is true, A || B evaluates to true. It evaluates to false only if both A and B are false.

{{end}} {{define "not-operator"}}

NOT Operator

The NOT operator, denoted by !, is a unary logical operator that negates the truth value of its operand. If the operand is true, ! returns false; if the operand is false, ! returns true.
Example: If condition A is true, !A evaluates to false. If A is false, !A evaluates to true.

{{end}}
